# OpenAI Service Design

아르바이트 권리 관리 AI 서비스 구현 프로젝트입니다. 기획서 기준으로 FE부터 시작했고, 이후 BE와 LLM 연동을 붙일 수 있도록 디렉토리를 분리했습니다.

## 디렉토리 구조

```text
FinalProject/
  FE/      정적 프론트엔드 프로토타입
  BE/      백엔드 API 구현 예정 영역
  docs/    기획, 실행 방법, 구현 메모
```

## FE 실행

현재 FE는 빌드 도구 없이 동작합니다. 아래 파일을 브라우저에서 열면 됩니다.

```text
FE/index.html
```

상세 구현 범위는 `docs/FE_PROTOTYPE.md`에 정리했습니다.
